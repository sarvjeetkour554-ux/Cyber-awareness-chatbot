# Cyber Awareness Assistant

## Overview

Cyber Awareness Assistant is an AI-powered chatbot designed to help users learn about cybersecurity threats and online safety. The chatbot provides simple and easy-to-understand answers related to phishing, ransomware, password security, UPI frauds, email scams, malware, safe browsing, and general cyber awareness.

## Features

* AI-powered cybersecurity assistance
* Supports English and Hindi responses
* Explains cyber threats in simple language
* Provides online safety tips
* User-friendly Streamlit interface
* Publicly accessible through a web link

## Technologies Used

* Python
* Streamlit
* Groq API
* HTML & CSS

## Topics Covered

* Phishing Attacks
* Password Security
* UPI Frauds
* Email Scams
* Malware
* Ransomware
* Social Engineering
* Safe Browsing
* Cyber Awareness
